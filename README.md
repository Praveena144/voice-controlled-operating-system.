Project Title
AI-Integrated Voice Controlled Operating System

Description
This project allows users to control the computer using voice commands.

Features
- Voice command recognition
- Hands-free interaction
- AI-based user behaviour learning
- Open applications using voice

Technologies
- Speech Recognition
- Natural Language Processing
- Machine Learning
- Python

- ## Technology Used
- Python
- AI
- Speech Recognition
- Natural Language Processing

## Project Objective
The objective of this project is to create a voice controlled operating system that allows users to control computer functions using voice commands.

## Output / Result
The system recognizes user voice commands and performs tasks like opening applications and controlling the system without using keyboard or mouse.
